# Mini-Project Web Page
    A responsive, modern web page showcasing hobbies like drones, photography, and more. Built with HTML and CSS for a clean, user-friendly experience.

# Features
1. Responsive Design: Works seamlessly on all screen sizes (desktop, tablet, mobile).
2. Grid Layout: Clean, organized sections for hobbies like drones, shooting, and more.
3. Media Integration: Responsive iframes and images for embedded content.
4. Mobile-Friendly: Vertical stacking and flexible layouts for smaller screens.
 
# Technologies Used
1. HTML5
2. CSS (Flexbox, Grid, Responsive Design)
3. JavaScript
